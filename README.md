# Web Scraper
Web Scraper tool with several available ports to scan.


<h2>Usage:</h2>
python3 webscraper.py (threads) (request rate, higher = faster scraping) (port)

<h2>Available ports</h2>
http (80, 8080)<br>
https (443)<br>
telnet (23)<br>
ftp (21)<br>
ssh (22)<br>
minecraft (25565)<br>
